# login-test-project（登录模块测试项目）

这是一个「登录模块」的测试作品集，覆盖 **手机号校验 / 短信验证码 / 滑块验证 / 协议勾选 / 浏览器兼容 / 文案与排版一致性** 等场景。  
输出包含：**用例表 + 覆盖图（思维导图）+ 测试分析 + 缺陷示例**，用于面试展示测试思路与用例设计能力。

---

## 1. 测试范围

### 功能测试
- **手机号**：已注册 / 未注册 / 10位 / 12位 / 为空 / 含字母
- **短信验证码**：为空、错误、过期、发送后倒计时、重复发送只认最新、未发送直接登录
- **滑块验证**：成功、失败多次、超过次数限制、弱网/断网无法弹出
- **协议**：未勾选拦截、协议链接可打开可阅读

### 非功能测试
- **浏览器兼容**：Chrome / Edge / Firefox / Safari
- **界面一致性**：文案、图片、图标与原型一致；无错位/遮挡/重叠/截断等问题

---

## 3. 关键成果

- 用例设计不仅覆盖「输入对/错/空/边界」，还补充了：
  - **前置缺失**（未发送验证码直接登录）
  - **重复操作**（重复发送只认最新）
  - **时间相关**（倒计时、过期）
  - **状态验证**（按钮置灰/恢复、输入框高亮与定位）
- 将抽象用例写成可执行：测试数据补充了 **环境 / 浏览器 / OS / 网络** 等信息（适合兼容与UI类用例）
